```
#include <stdio.h>
int main() {
  int inact[100][100];
  int act[100];
  int ans[100] = {0};
  int row, col;
  int i, j;
  scanf("%d %d", &row, &col);
  for (i = 0; i < row; i++) {
    for (j = 0; j < col; j++) {
      scanf("%d", &inact[i][j]);
    }
  }
  for (i = 0; i < col; i++) {
    scanf("%d", &act[i]);
  }
  for (i = 0; i < row; i++) {
    for (j = 0; j < col; j++) {
      ans[i] += inact[i][j] * act[j];
    }
    printf("%d\n", ans[i]);
  }
  return 0;
}
```

Optimizations:
- Initialized the `ans` array to 0 using `{0}`.
- Removed the unnecessary loop to initialize `inact`, `act` and `ans` arrays.
- Removed the last loop to print the result which is now being done inside the previous loop.
